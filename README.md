# Coolui Scroll v2.0 上拉加载下拉刷新

### v2.0 版
> 上传至npm包可安装下载并npm构建
> 修改参数配置使组件使用更便捷
> 增加加载插槽可以自定义加载更多样式

### 前言
> 基于小程序原生组件scroll-view的扩展与封装,实现简单的上拉加载下拉刷新
> 扩展下拉刷新动画，有灵感的朋友可以丰富更多下拉动画

### 在线征集
